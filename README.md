# Boza - Pantalla de Login

Pantalla de inicio de sesión en HTML, CSS y JavaScript usando la paleta de colores proporcionada.

## Paleta de Colores
- Glaucous: `#59789F`
- Powder Blue: `#A9B6C4`
- Vanilla: `#ECE69D`
- Moss Green: `#7A9445`
- Dark Green: `#243C2C`
- Logo Brown (aprox.): `#4A2F1B`

## Estructura
```
index.html
styles.css
script.js
assets/ (poner aquí logo.png)
```

## Uso
1. Copia el logo en `assets/logo.png`.
2. Abre `index.html` en el navegador.
3. Ingresa correo y contraseña. Para prueba rápida la contraseña válida simulada es `boza123`.
